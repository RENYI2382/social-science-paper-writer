# 社科与社工论文正文写作助手 
(Social Science Paper Writer)

这是一个专门从“论文全流程工作台”中剥离出来的独立项目，聚焦于**论文正文阶段的撰写、扩写与改写**。

它摒弃了文献检索、阅读等前置环节，专注于解决一个问题：**如何把大白话初稿、访谈记录、粗糙的提纲，变成一段严谨、克制、富有机制解释力的社会科学论文正文。**

## 核心能力
1. **去情绪化，增强机制解释**：把“令人动容的悲怆”改写为“在特定制度安排下的资源剥夺”。
2. **理论内嵌式叙述**：不再机械地“张三认为、李四认为”，而是把文献与理论直接揉碎融合在对经验材料的分析中。
3. **专业社工实务写作**：写出社工的“专业判断”与“为何如此决策”，而不是仅仅记录“做了什么”的过程流水账。
