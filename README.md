# 社科与社工论文：正文写作与精修助手

这是一个专门针对**学术论文正文部分**的写作与精修工具（注意：文献综述的写法不在本项目范围内）。

无论你是使用 ChatGPT、Kimi、Claude，还是使用专业的 Trae IDE，这个项目都能帮你把**大白话初稿、零散的田野笔记、粗糙的机翻段落**，瞬间重构为符合学术规范、严谨克制、且富有机制解释力的正文段落。

---

## 🎯 应该喂给 AI 什么？（极其重要）

为了让论文改写的质量最高，**千万不要把整篇一万字的论文一次性扔给 AI**。AI 无法一次性处理宏大的逻辑。

你需要一段一段（或一节一节）地来。最佳的“喂食”材料是：

1. **粗糙的正文草稿（500 - 1500 字）**
   * *比如：“我大概写了一段关于留守儿童怎么使用手机的现象，但写得像记叙文，缺乏学术感。”*
2. **零散的访谈记录与田野笔记**
   * *比如：“这是我今天跟案主张大爷的访谈记录原话，我需要把它写成一段带社工专业判断的实务分析段落。”*
3. **僵硬的机翻/拼接段落**
   * *比如：“这段理论论述看起来很生硬，全在罗列‘张三认为...李四认为...’，我需要把它融合得更像人写的。”*

## � 极简使用指南（适合所有网页端 AI 用户）

如果你不知道什么是 Skill，也没关系。你只需要**复制下面的提示词（Prompt）模板，粘贴到你常用的 AI 工具（如 Kimi、ChatGPT）中，并在最下方贴上你的草稿即可**。

根据你的专业，选择以下两个模板之一：

### 模式一：通用社科模式（适合政治学、管理学、教育学、社会学等）

**👉 复制以下全部内容，发给 AI：**

```text
你现在是一位在社会科学领域具有深厚造诣的资深学术研究者与写作顾问。
请帮我改写/扩写以下论文正文草稿。

【核心写作哲学】
1. 解释机制而非抒发情感：拒绝文学化表达（如“令人动容”“悲怆”）。将所有现象转化为“条件、机制和后果”的分析。
2. 结构化经验分析：在分析问题时，采用“制度条件—角色分工—话语环境”的联动分析框架，揭示个体实践背后的社会结构性因素。
3. 理论内嵌：不要使用“张三认为、李四认为”的罗列式写法。将理论和文献自然地融合在对经验材料的分析句子中。
4. 段落与小标题：不可以使用罗列式（1. 2. 3.）论述，必须采用逻辑严密的展开段落式叙述。小标题必须是一句简洁、严谨的陈述句。

【我的草稿/访谈材料】：
（请在这里粘贴你 500-1500 字的草稿或笔记）
```

### 模式二：社会学与社会工作实务专属模式

**👉 复制以下全部内容，发给 AI：**

```text
你现在是一位社会学/社会工作领域的资深学术研究者。
请帮我把以下实务案例/访谈记录，改写为符合社会工作高标准期刊的正文分析段落。

【核心写作哲学与社工伦理】
1. 机制解释：不要写“从社会学视角审视”这种外行话。你要直接写出：在什么制度安排下，谁被要求承担风险，谁获得资源，结果如何反复出现。
2. 实务分析规则：案例并非目的，而是展示专业判断的媒介。不要仅复现“做了什么”（过程记录），而要回答“为何如此决策而非他法”（专业决策逻辑）。当案例材料被抽离时，文本中仍应清晰保留问题建构的逻辑。
3. 学术伦理：引用案主的原话要保留并呈现数据，但案主必须使用化名（如：案主A）。
4. 行文规范：除非是引文或术语界定，正文中不使用双引号。深入浅出，不要过度堆砌晦涩术语来掩盖案例的鲜活度。

【我的实务记录/访谈材料】：
（请在这里粘贴你的实务过程记录、访谈稿或初稿片段）
```

## � 高阶使用指南（适合 Trae / VSCode 等 IDE 用户）

如果你在使用 Trae 等支持 MCP 与 Skill 的智能 IDE：

1. 将本项目的整个文件夹（`social-science-paper-writer`）放入你论文工作区的 `.trae/skills/` 目录下。
2. 唤醒 Trae 并输入：`调用正文写作助手，帮我把这段访谈记录改成社工实务分析段落。`
3. 助手会自动读取 `prompts/` 目录下的底层规则文件（`01_general_social_science_writer.txt` 或 `02_sociology_and_social_work_writer.txt`），并直接在你的编辑器侧边栏为你生成高质量的学术正文。

## � 效果对比示例

**你给 AI 的大白话初稿：**
> “今天去了社区，发现很多老人不会用智能手机，去医院看病连号都挂不上，真的很让人心酸。作为社工，我赶紧帮他们办了就诊卡，还教他们怎么用微信挂号，老人们都很感激我，这体现了社工在社区养老里的重要作用。”

**经过【模式二：社工实务模式】改写后的正文段落：**
> “在当前的数字医疗资源分配体系中，技术壁垒无形中剥夺了部分老年群体的基本医疗可及性。面对这一结构性排斥，社会工作者的介入并非单纯的技术替代，而是作为‘资源链接者’重建案主的社会支持网络。在本案中，研究者并未仅停留在代为挂号的工具性服务层面，而是通过示范与培力（Empowerment），试图恢复案主在数字化生存环境中的主体性。这一行动决策的逻辑在于：若仅提供替代性服务，将进一步固化案主的弱势地位；唯有将其转化为技能赋权，方能触及社区养老服务供给的深层困境。”
