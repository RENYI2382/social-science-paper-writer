# 社科与社工论文正文写作助手 
(Social Science Paper Writer)

这是一个专门针对**学术论文正文阶段**的撰写与精修工具。

它的核心目的是：把大白话初稿、零散的田野访谈笔记、粗糙的提纲，重构为符合学术规范、严谨克制、且富有机制解释力的正文段落。

---

## 🛠️ 双轨工作模式

为了更精准地服务不同专业背景的研究，本项目拆分为两个完全独立的工作模块。请在使用时根据你的具体专业选择对应的助手：

### 模式一：通用社科论文助手
> 📄 **对应 Prompt:** `prompts/01_general_social_science_writer.txt`
> 🎯 **适用对象:** 政治学、管理学、教育学、传播学等广泛的社会科学研究。

**核心能力：**
- **机制解释导向**：强制将描述性、抒情性文本转化为“条件—机制—后果”的结构化分析。
- **去情绪化**：剔除所有“令人动容”、“悲怆”等主观态度词，恢复学术克制。
- **理论内嵌**：避免机械的“张三认为、李四认为”，将文献作为默认知识自然地融入论述段落中。

### 模式二：社会学与社会工作专业助手
> 📄 **对应 Prompt:** `prompts/02_sociology_and_social_work_writer.txt`
> 🎯 **适用对象:** 专精于社会学、社会工作实务、人类学等需要深度处理经验材料的学科。

**进阶能力 (在通用模式基础上增强)：**
- **社会学透视**：强制禁止使用“从社会学视角看”这种外行词汇。直接通过分析“在什么制度安排下，谁承担风险，谁获得资源”来呈现真正的社会学品味。
- **实务案例重构**：专门针对社工实务材料。要求案例描写必须服务于“专业判断”，凸显研究者的决策逻辑（为何如此而非他法），拒绝单纯复述“做了什么”的过程流水账。
- **严厉的学术伦理**：强制规范访谈文本的处理方式（案主化名、原话保留、禁用滥用双引号等）。

## 🚀 如何使用

1. 将本文件夹放入工作区的 `.trae/skills/` 目录下。
2. 唤醒 Trae 并调用本 Skill。
3. 告诉助手你的身份：
   - *“我有一段粗糙的初稿，请用通用社科模式帮我改写，增强机制感。”*
   - *“这是一段社工访谈记录，请用社会工作模式帮我提炼出实务背后的专业决策逻辑，写成正式的论述段落。”*
4. 粘贴你的草稿或笔记，等待高质量的学术正文产出。
